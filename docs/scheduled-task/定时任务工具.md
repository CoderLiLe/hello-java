# 定时任务工具

## 1. 定时任务对比

| 工具 | 架构 | 支持分布式 | 持久化 | 管理界面 |
|------|------|-----------|--------|---------|
| Quartz | 中心化 | 需 JDBC | 是 | 无 |
| XXL-JOB | 中心化 | 是 | 是 | 有 |
| ElasticJob | 去中心化 | 是 | 是 | 有 |
| Spring Task | 单机 | 否 | 否 | 无 |

## 2. Spring Task

### 2.1 简单使用
```java
@EnableScheduling
@SpringBootApplication
public class Application {}

@Component
public class MyTask {
    @Scheduled(cron = "0 0/5 * * * ?")  // 每 5 分钟
    public void run() {
        System.out.println("定时任务执行: " + new Date());
    }
    
    @Scheduled(fixedRate = 5000)         // 每 5 秒（上次开始算）
    @Scheduled(fixedDelay = 5000)        // 每 5 秒（上次结束算）
    @Scheduled(initialDelay = 10000, fixedRate = 5000)  // 延迟 10 秒
    public void anotherTask() {}
}
```

### 2.2 Cron 表达式
```
秒 分 时 日 月 周
0  0  12 * * ?   -> 每天 12:00
0 15 10 ? * *    -> 每天 10:15
0 0/5 14 * * ?   -> 14 点起每 5 分钟
0 0 2 ? * MON-FRI -> 工作日凌晨 2 点
```

## 3. Quartz

### 3.1 核心概念
- **Job**：任务接口
- **JobDetail**：任务定义
- **Trigger**：触发器（SimpleTrigger / CronTrigger）
- **Scheduler**：调度器

### 3.2 使用
```java
// 定义 Job
public class MyJob implements Job {
    @Override
    public void execute(JobExecutionContext context) {
        System.out.println("执行任务: " + context.getJobDetail().getKey());
    }
}

// 调度
JobDetail job = JobBuilder.newJob(MyJob.class)
    .withIdentity("myJob", "group1")
    .build();

Trigger trigger = TriggerBuilder.newTrigger()
    .withIdentity("myTrigger", "group1")
    .withSchedule(CronScheduleBuilder.cronSchedule("0/5 * * * * ?"))
    .build();

Scheduler scheduler = StdSchedulerFactory.getDefaultScheduler();
scheduler.start();
scheduler.scheduleJob(job, trigger);
```

## 4. XXL-JOB

### 4.1 架构
```
调度中心（Admin）
    |
执行器（Executor）- 执行器（Executor）- 执行器（Executor）
```

### 4.2 使用
```java
@Component
public class MyJobHandler {
    @XxlJob("myJobHandler")
    public void execute() {
        XxlJobHelper.log("任务开始");
        // 业务逻辑
        XxlJobHelper.log("任务结束");
    }
}
```

## 5. 分布式任务注意事项

- **幂等性**：保证重复执行结果一致
- **分片**：数据分片处理（如按 userId 取模）
- **失败重试**：合理设置重试次数
- **告警**：任务失败通知
- **日志**：执行日志完整记录
