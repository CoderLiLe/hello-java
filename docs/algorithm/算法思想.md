# 常见应用级算法思想

## 1. 时间复杂度与空间复杂度

### 1.1 大 O 表示法
- O(1)：常数阶，哈希表查找
- O(log n)：对数阶，二分查找
- O(n)：线性阶，单层循环
- O(n log n)：线性对数阶，归并/快排
- O(n²)：平方阶，双层循环
- O(2ⁿ)：指数阶，递归斐波那契

### 1.2 空间复杂度
- O(1)：原地操作
- O(n)：额外数组/链表
- O(n²)：二维数组

## 3. 查找算法

### 3.1 二分查找
```java
int binarySearch(int[] arr, int target) {
    int left = 0, right = arr.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (arr[mid] == target) return mid;
        if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
```

## 4. 常用算法思想

### 4.1 贪心算法
每步选择局部最优，最终达到全局最优。
- 经典应用：活动选择、哈夫曼编码、最小生成树

### 4.2 分治算法
```java
// 归并排序
void mergeSort(int[] arr, int left, int right) {
    if (left < right) {
        int mid = left + (right - left) / 2;
        mergeSort(arr, left, mid);
        mergeSort(arr, mid + 1, right);
        merge(arr, left, mid, right);
    }
}
```

### 4.3 动态规划
将问题拆分为子问题，记录子问题结果避免重复计算。

```java
// 斐波那契
int fib(int n) {
    if (n <= 1) return n;
    int[] dp = new int[n + 1];
    dp[0] = 0; dp[1] = 1;
    for (int i = 2; i <= n; i++) {
        dp[i] = dp[i-1] + dp[i-2];
    }
    return dp[n];
}

// 0-1 背包
int knapsack(int W, int[] wt, int[] val) {
    int n = val.length;
    int[][] dp = new int[n+1][W+1];
    for (int i = 1; i <= n; i++) {
        for (int w = 1; w <= W; w++) {
            if (wt[i-1] <= w) {
                dp[i][w] = Math.max(dp[i-1][w], dp[i-1][w-wt[i-1]] + val[i-1]);
            } else {
                dp[i][w] = dp[i-1][w];
            }
        }
    }
    return dp[n][W];
}
```

## 5. 业务场景算法

### 5.1 布隆过滤器 Bloom Filter
- 判断元素"一定不存在"或"可能存在"
- 位图 + 多个哈希函数
- 应用：防止缓存穿透、垃圾邮件过滤

### 5.2 一致性哈希 Consistent Hash
- 解决分布式哈希表的动态伸缩问题
- 虚拟节点提高均衡性
- 应用：Redis 集群、负载均衡

### 5.3 限流算法
- **计数器**：固定窗口计数
- **滑动窗口**：细化窗口
- **漏桶**：固定速率处理
- **令牌桶**：允许一定突发

### 5.4 限流实现（令牌桶）
```java
public class TokenBucket {
    private final long capacity;     // 桶容量
    private final long refillRate;   // 速率
    private final long refillInterval;
    private long tokens;
    private long lastRefillTime;
    
    public synchronized boolean tryAcquire() {
        refill();
        if (tokens > 0) {
            tokens--;
            return true;
        }
        return false;
    }
    
    private void refill() {
        long now = System.currentTimeMillis();
        long elapsed = now - lastRefillTime;
        long addTokens = elapsed / refillInterval * refillRate;
        tokens = Math.min(capacity, tokens + addTokens);
        lastRefillTime = now;
    }
}
```

### 5.5 LRU 缓存
```java
class LRUCache<K, V> extends LinkedHashMap<K, V> {
    private final int capacity;
    
    public LRUCache(int capacity) {
        super(capacity, 0.75f, true);
        this.capacity = capacity;
    }
    
    public V get(Object key) { return super.getOrDefault(key, null); }
    
    @Override
    protected boolean removeEldestEntry(Map.Entry<K, V> eldest) {
        return size() > capacity;
    }
}
```
